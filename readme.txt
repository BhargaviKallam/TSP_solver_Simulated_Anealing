		Travelling Salesman Problem Solver with Simulated Annealing(SA)

The packages used are menthioned below alongside with version.

• Python == 3.8.8
• matplotlib == 3.3.4
• NumPy == 1.19.5

The source code for this assignment is implemented under Python, Notepad++.

	How to execute:

		Run : $python tsp_solver.py filename.tsp

This extracts the specific file give in command from the tsplib library using url path pre-defined in the source code. Also, generates a fitness graph automatically.

		Run : $type solver.csv

This command outputs the result optimal_tour(coordinates) that is stored into .csv file in the current directory.
